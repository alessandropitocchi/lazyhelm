## How to Contribute

1. Fork the repo
2. Create a feature branch
3. Make your changes
4. Add tests if applicable
5. Run `go test ./...`
6. Submit a pull request

## Code Style

- Follow standard Go conventions
- Run `go fmt` before committing
- Write clear commit messages
